# 余光中 散文 voice profile（初版）

## 風格摘要

短句鎚定音、長句鋪意象，靠疊字與擬聲讓聲音自己說話。古典詞與白話交織不違和；動詞密集成鏈，不用抽象包裝。情感主軸常是時間流逝與鄉愁，但不直說，靠具體意象（雨、瓦、傘、巷）反覆敲擊。

---

## 可執行寫法規則

### 1. 疊字節奏優先

同一意象可用 ABAB / AABB / ABCABC 多型疊字延伸，不平鋪直敘。

| 平的 | 余光中 |
|---|---|
| 春寒漸增 | 料料峭峭 |
| 雨下得急 | 淋淋漓漓 ／ 淅淅瀝瀝 |
| 地很濕 | 潮潮地濕濕 |
| 思緒受潮 | 連思想也都是潮潤潤的 |

### 2. 聽覺擬聲直接入文

雨聲、雷聲、蟲聲、節奏類**直接擬聲**，不只描述「下雨」「打雷」。

| 平的 | 余光中 |
|---|---|
| 暴雨打來 | 滔天的暴雨滂滂沛沛撲來 |
| 雷聲很響 | 強勁的電琵琶忐忐忑忑忐忐忑忑 |
| 葉子聲 | 叢葉嘈嘈切切 |
| 蠶吃桑葉 | 細細瑣瑣屑屑，口器與口器咀咀嚼嚼 |

### 3. 古典白話交織

同段內混用「驚蟄」「料峭」「孺慕」「皇天后土」「皚皚」古典詞，與「他想」「曲折穿過」「不過那塊土地」白話。**不專文白、不專口語，互相托住。**

| 平的 | 余光中 |
|---|---|
| 春天到了 | 驚蟄一過，春寒加劇 |
| 想念家鄉 | 不能撲進她懷裡，被她的裙邊掃一掃也算是安慰孺慕之情吧 |

### 4. 句長強烈對照

中段堆長句鋪意象，結尾用 5–8 字短句一鎚收口。

範例：
> 「在古老的大陸上，千屋萬戶是如此。二十多年前，初來這島上，日式的瓦屋亦是如此。先是天黯了下來，城市像罩在一塊巨幅的毛玻璃裡，陰影在戶內延長復加深。」（鋪陳）
>
> **「前塵隔海。古屋不再。聽聽那冷雨。」**（鎚收口）

### 5. 動詞密集鏈

一個動作切成多個短動詞並列推進，不用「進行 X」「作出 Y」抽象包裝。

| 平的 | 余光中 |
|---|---|
| 走過彎曲的小巷 | 曲折穿過金門街到廈門街迷宮式的長巷短巷，雨裡風裡，走入霏霏令人更想入非非 |
| 把濕雨變成雪 | 索性把濕濕的灰雨凍成乾乾爽爽的白雨 |
| 雨珠四濺 | 將骨柄一旋，雨珠向四方噴濺，傘緣便旋成了一圈飛簷 |

### 6. 拒絕英式中文（負規則）

與 writing-principles.md 同源：不用「被」字被動、少「的」字三連、不堆抽象名詞、不寫「進行討論／作出決定」。**這條是底線，與 anti-AI floor rules 5/7/8 同等強度。**

## 神韻 sample 段落（從〈聽聽那冷雨〉擷取）

**Sample A — 疊字 + 擬聲 + 古典白話交織**

> 驚蟄一過，春寒加劇。先是料料峭峭，繼而雨季開始，時而淋淋漓漓，時而淅淅瀝瀝，天潮潮地濕濕，即連在夢裡，也似乎有把傘撐著。

**Sample B — 動詞鏈 + 短句鎚收**

> 雨打在樹上和瓦上，韻律都清脆可聽。尤其是鏗鏗敲在屋瓦上，那古老的音樂，屬於中國。……
>
> **前塵隔海。古屋不再。聽聽那冷雨。**

**Sample C — 跨段 motif 呼喚**

> 「聽聽，那冷雨。看看，那冷雨。嗅嗅聞聞，那冷雨，舔舔吧，那冷雨。」
>
> 標題式短語反覆呼喚，形成節奏複沓。/write 套此 voice 時可在段落首句插入 motif 變奏，但**全篇不超過 3 次**，避免變成裝飾性排比（rule 7 仍守）。

## 詞彙線索

**古典詞匯**：驚蟄、料峭、孺慕、皇天后土、皚皚、虛虛幻幻、纏纏綿綿、霏霏、迢迢、摧心折骨、宵寒袭肘

**動詞輕重**（偏好有重量的動詞）：壓、敲、鎚、鞭、灑、漫、滲、侵、撲、奏、彈、叩、噴濺、咀嚼

**疊字模板**：
- ABAB：料料峭峭、忐忐忑忑、淋淋漓漓
- AABB：清清醒醒、虛虛幻幻、霏霏不絕
- ABCABC：細細瑣瑣屑屑
- 四字成語式：杏花春雨、殘山剩水、皇天后土、千山萬山

**西文嵌入**（克制使用）：rain ／ pluie ／ 安東尼奧尼 ／ 安格羅薩克遜——做意象對照，**不喧賓奪主**。

**數字時間軸**：「二十五年」「四分之一的世紀」「一打少年聽雨」「再打中年聽雨」「三打白頭聽雨」疊砌時間流逝。

## 反面（套此 voice 時同步守 anti-AI floor）

此 preset 是**正向風格錨**，仍須守 zh rules 5 / 7 / 8 anti-AI 味底線（禁對仗句 ／ 禁排比 ／ 禁名詞化）。/write 主流程已強制此底線，本 preset 不覆蓋。

額外避免（與余光中本人寫作論一致）：

- 「被」字被動式
- 「的」字三連
- 「進行 X」「作出 Y」抽象動詞包裝
- 空泛形容詞（「美麗的 X」單用無感官錨）
