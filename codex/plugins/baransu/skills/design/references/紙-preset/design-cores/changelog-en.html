Changelog

This file follows Semantic Versioning (MAJOR.MINOR.PATCH) and the Keep a Changelog convention.

  • 1.4.0

    2026-05-13

    Schema suite complete: eight document schemas across three presets and two languages, plus Kami chevron and two-tier node-width discipline.

    Added

    • Equity-Report schema (Thesis / Valuation / Risk plus SVG quadrant figure).
    • Changelog schema (semantic version plus Added / Changed / Removed / Fixed).
    • Three presets times two languages: twelve new HTML templates.

    Changed

    • editorial-sanity.sh dropcap window widened to 4.0em–5.0em (was 4.5em–4.85em).
    • validate-output.ts GATE-J narrows node-width whitelist to {128, 144, 160}.

    Removed

    • Legacy polygon-based marker patterns are no longer supported.

    Fixed

    • Letter en variant: Charter fallback order corrected.
    • Resume zh template: missing object-position caused portrait drift.

  • 1.3.0

    2026-04-02

    Introduce class-prefix whitelist across three presets (kami / swiss / google); add GATE-J and GATE-K gates.

    Added

    • GATE-J node-width-whitelist (max two tiers).
    • GATE-K chevron-strict (marker path must be M2 1 L8 5 L2 9).

    Changed

    • Three preset sanity scripts now delegate to a shared editorial-sanity.sh.

    Removed

    • Experimental gd-* class prefix (inconsistent with the long-form template).

    Fixed

    • One-Pager @page margins overflowed the printable area on US Letter paper.
