Design System: 紙 (Kami)

溫潤羊皮紙感。介面如一張好紙——有重量、有肌理、讓文字安靜地躺在上面。書房、工藝、手感印刷品；不是高科技冷峻，是紙墨溫度。

1. Visual Theme & Atmosphere

哲學是 warm printed paper — ink on parchment, craft over chrome。Kami 是一套印刷學優先的設計語言，背景永遠是暖調羊皮紙、accent 永遠是墨藍、字型永遠是 serif。十條不變量約束視覺一致：

  1. 背景不用純白，用暖調羊皮紙 #f5f4ed
  2. Accent 唯一墨藍 #1B365D，全頁只此一個彩色
  3. 中性灰限定暖調（黃棕底色），禁冷灰
  4. 每頁只用一套 serif；--font-sans 永遠 alias --font-serif
  5. Serif weight 鎖定 500——禁用 700/bold 做 heading
  6. 行高：headline 1.1–1.3 / 密排 1.4–1.45 / 閱讀 1.5–1.55
  7. 字距：中文 0.1–0.3pt；英文 body 0；label/cap +0.2–1pt
  8. Tag 背景只用 solid hex（禁 rgba，WeasyPrint 渲染 bug）
  9. 深度只用 ring / whisper 陰影，禁硬邊陰影
  10. 禁 italics（模板與 demo 均適用）

2. Color Palette & Roles

3. Typography

英文 stack Charter, Georgia, Palatino, "Times New Roman", serif；中文 stack TsangerJinKai02, "Source Han Serif SC", "Noto Serif CJK SC", "Songti SC", "STSong", Georgia, serif。--font-sans intentionally aliases --font-serif；不引入獨立 sans 字族。

5. Layout & Spacing

間距基底 4pt（--space-xs … --space-3xl）；radius scale 4pt → 6pt → 8pt → 12pt → 16pt → 24pt → 32pt (hero)。12 欄網格、column gutter 24px、page margin 48px（桌機）/ 16px（手機）。閱讀欄位最大 680px；全版面容器最大 1200px。

  • 內容區塊間距至少 32px
  • 標題下方 16px、上方 48px
  • 卡片內 padding 20–24px

6. Iconography & Imagery

圖示線條型 outline，stroke-width 1.5–2px，圓角端點；尺寸 16 / 20 / 24px 三檔；顏色繼承 currentColor，特殊強調才用 --accent。圖像優先低飽和暖調與紙質感，避免高彩度商業 stock。Data chart 色序：#1B365D → #504e49 → #6b6a64 → #b8b7b0 → #d4d3cd → #EEF2F7。

7. Motion

過渡時間：micro 150ms / 狀態切換 250ms / 頁面 350ms。Easing 標準 ease-out；彈性 cubic-bezier(0.34, 1.56, 0.64, 1)。不使用裝飾性循環動畫，尊重 prefers-reduced-motion；優先 opacity + scale，避大範圍平移。

8. Do / Don’t

Do

  • 暖調羊皮紙 #f5f4ed 作主背景
  • 墨藍 #1B365D 為唯一強調色（≤5% 表面）
  • Charter / TsangerJinKai02 serif 堆疊
  • Heading weight 鎖 500，不用 700/bold
  • 陰影僅 ring / whisper 兩種
  • Body 行高 1.5–1.55，headline 1.1–1.3
  • Tag 背景 solid hex #EEF2F7
  • --font-sans 永遠 alias --font-serif
  • Section title 用 2.5pt brand left bar

Don’t

  • 禁用純白背景 #ffffff
  • 禁用冷灰 / 藍調 neutral
  • 禁硬邊陰影（blur < 4px）
  • 禁 tag/badge rgba 背景
  • 禁用 bold (700) heading
  • 禁 italics（模板與 demo 均適用）
  • 禁全頁無襯線字族
  • 禁高飽和度撞色
  • 禁裝飾性循環動畫

9. AI Prompt Guide

以下 reproducer prompt 可在全新 AI 對話中重現本設計系統視覺語言：

Design a UI using the Kami paper design system. Background: warm parchment #f5f4ed; card surfaces: #faf9f5; interactive surfaces: #e8e6dc. Primary accent: ink-blue #1B365D (≤5% surface area). Text hierarchy: near-black #141413 → warm dark #3d3d3a → olive #504e49 → stone #6b6a64. Borders: --border: #e8e6dc (primary), --border-soft: #e5e3d8 (secondary). Typography: Charter/Georgia for English, TsangerJinKai02/Noto Serif SC for Chinese — all weights locked at 500 for headings (no bold), 400 for body. --sans aliases --serif. Line-height: headlines 1.1–1.3, body 1.5–1.55. Shadows: ring only (0 0 0 1px var(--border)) for static; whisper (0 4px 24px rgba(0,0,0,0.05)) for hover. No hard drop shadows. Tags use solid hex #EEF2F7 (never rgba). Section titles use a 2.5pt brand left bar. No italics anywhere. The aesthetic is warm printed paper — ink on parchment, craft over chrome.

(a) 焦點節點上限

每一頁、每一張投影片，焦點節點上限為 1–2 個。主焦點以 --accent: #1B365D 染色（H1、品牌左 bar、唯一 CTA）；次焦點以「結構強度」承擔（加粗 metric 數字、明顯 left-bar 分節線），不得再用 accent。違反即視為色彩過載。

(b) accent hex 設計理據

SpaceCoordinates設計意圖
HEX#1B365D主規格；WeasyPrint print pipeline 以此為準
HSLH 211°, S 55%, L 24%冷色相位 211°；S 55% 避過飽和；L 24% 為深色印刷藍——羊皮紙上高對比但不刺眼
oklchoklch(0.32 0.08 256)screen / 設計工具 advisory；色相 256° perceptual 等價；C 0.08 對應印刷藍低彩度

選色理由：對齊鋼筆墨水在 cream paper 上的視覺記憶；避開飽和 royal blue（過於數位）與 navy（過於企業）兩端。

(c) 我不是什麼（allowed contradictions）

  • no second accent — 全頁只一個彩色；任何「次強調色」皆以中性灰階重量替代
  • no italics — 模板與 demo 均禁用，襯線斜體在 print pipeline 容易掉字
  • no cool accent shift — accent 不得偏向 cyan / teal；warm parchment 需要 warm-shifted blue
  • no oklch in attribute — tokens.css / inline style 不得出現 oklch(；oklch 僅為 advisory
  • no gradient bg — 背景永遠平塗紙色；禁 linear-/radial-gradient 背景
  • no hard drop shadow — 深度只用 ring + whisper；禁 0 8px 16px rgba(0,0,0,0.3) 類硬邊陰影
  • no all-caps headings — 中英混排不使用 letter-spacing + uppercase 的設計派頭
