Design System: 紙 (Kami)

溫潤羊皮紙感。介面如一張好紙——有重量、有肌理、讓文字安靜地躺在上面。書房、工藝、手感印刷品；不是高科技冷峻，是紙墨溫度。

1. Visual Theme & Atmosphere

哲學是 warm printed paper — ink on parchment, craft over chrome。Kami 是一套印刷學優先的設計語言，背景永遠是暖調羊皮紙、accent 永遠是墨藍、字型永遠是 serif。十條不變量：

  1. 背景不用純白，用暖調羊皮紙 #f5f4ed
  2. Accent 唯一墨藍 #1B365D，全頁只此一個彩色
  3. 中性灰限定暖調，禁冷灰
  4. 每頁只用一套 serif；--font-sans alias --font-serif
  5. Serif weight 鎖 500，禁 700/bold heading
  6. 行高 headline 1.1–1.3 / 密排 1.4–1.45 / 閱讀 1.5–1.55
  7. 字距 中文 0.1–0.3pt；英文 body 0；label +0.2–1pt
  8. Tag 背景 solid hex（禁 rgba，WeasyPrint bug）
  9. 深度只用 ring / whisper 陰影，禁硬邊
  10. 禁 italics（模板與 demo 均適用）

5. Layout & Spacing

間距 4pt 基底（--space-xs … --space-3xl）；radius scale 4 → 6 → 8 (預設) → 12 → 16 → 24 → 32 (hero)。12 欄網格、column gutter 24px、page margin 48px / 16px。閱讀欄位最大 680px；全版面最大 1200px。

  • 內容區塊間距至少 32px
  • 標題下方 16px、上方 48px
  • 卡片內 padding 20–24px

6. Iconography & Imagery

圖示線條型 outline，stroke 1.5–2px；尺寸 16 / 20 / 24px；顏色繼承 currentColor。圖像優先低飽和暖調、紙質感。Data chart 色序 #1B365D → #504e49 → #6b6a64 → #b8b7b0 → #d4d3cd → #EEF2F7。

7. Motion

過渡 micro 150ms / 狀態 250ms / 頁面 350ms。Easing 標準 ease-out；彈性 cubic-bezier(0.34, 1.56, 0.64, 1)。不裝飾性循環動畫；尊重 prefers-reduced-motion。

8. Do / Don’t

Do

  • 暖羊皮紙 #f5f4ed 主背景
  • 墨藍 #1B365D 唯一強調色（≤5%）
  • Charter / TsangerJinKai02 serif 堆疊
  • Heading weight 500，不用 700
  • 陰影僅 ring / whisper
  • Tag 用 solid hex #EEF2F7
  • Section title 用 2.5pt brand left bar

Don’t

  • 禁純白背景
  • 禁冷灰 / 藍調 neutral
  • 禁硬邊陰影（blur < 4px）
  • 禁 tag rgba 背景
  • 禁 bold (700) heading
  • 禁 italics
  • 禁無襯線全頁排版

9. AI Prompt Guide

以下 reproducer prompt 可在全新 AI 對話中重現本設計系統：

Design a UI using the Kami paper design system. Background: warm parchment #f5f4ed; card surfaces: #faf9f5; interactive surfaces: #e8e6dc. Primary accent: ink-blue #1B365D (≤5% surface area). Text hierarchy: near-black #141413 → warm dark #3d3d3a → olive #504e49 → stone #6b6a64. Typography: Charter / Georgia for English, TsangerJinKai02 / Noto Serif SC for Chinese — all weights locked at 500 for headings (no bold), 400 for body. --font-sans aliases --font-serif. Line-height: headlines 1.1–1.3, body 1.5–1.55. Shadows: ring only (0 0 0 1px var(--border)); whisper (0 4px 24px rgba(0,0,0,0.05)) for hover. No hard drop shadows. Tags use solid hex #EEF2F7 (never rgba). Section titles use a 2.5pt brand left bar. No italics anywhere. The aesthetic is warm printed paper — ink on parchment, craft over chrome.
