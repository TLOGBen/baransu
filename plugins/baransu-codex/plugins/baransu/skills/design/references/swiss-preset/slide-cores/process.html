How the loop runs

  1. 01

    Frame

    State the user problem in one sentence.

  2. 02

    Probe

    Pull telemetry, interview three users.

  3. 03

    Draft

    Sketch the smallest viable response.

  4. 04

    Ship

    Release behind a flag, watch for drift.

  5. 05

    Reflect

    Retire the flag or repeat the loop.
